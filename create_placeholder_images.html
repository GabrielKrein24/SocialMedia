Imagens Placeholder para o Projeto Conect+

Para adicionar as imagens reais ao projeto:

  1. Hero Image (hero-conect.jpg):
    • Salve a segunda imagem fornecida como public/images/hero-conect.jpg
    • Dimensões recomendadas: 1200x800px
    • Formato: JPEG otimizado (qualidade 85%)
    • Alt text: "Equipe da Conect+ celebrando resultados de marketing nas redes sociais"
  2. About Image (about-conect.webp):
    • Salve a primeira imagem fornecida como public/images/about-conect.webp
    • Dimensões recomendadas: 1200x800px
    • Formato: WebP otimizado (qualidade 85%)
    • Alt text: "Apresentação da Conect+ com gráficos de desempenho em tela"

Comandos para Otimização (se necessário)


# Para converter para WebP (usando imagemagick ou cwebp)
magick hero-image.jpg -quality 85 -resize 1200x800^ public/images/hero-conect.jpg
cwebp -q 85 -resize 1200 800 about-image.jpg -o public/images/about-conect.webp

# Para otimizar JPEG
jpegoptim --size=500k public/images/hero-conect.jpg
