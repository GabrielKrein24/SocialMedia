# Conect+ - Agência de Marketing Digital

Site moderno e otimizado para a agência Conect+, especializada em marketing digital e gestão de mídias sociais.

### Tipografia
- **Fonte**: Inter (Google Fonts)

## 🏗️ Estrutura do Site

### Seções Principais
1. **Header**: Logo, menu de navegação e CTA
2. **Hero Section**: Título impactante, descrição e call-to-actions
3. **Serviços**: Cards animados com ícones e descrições
4. **Prova Social**: Depoimentos, métricas e logos de clientes
5. **Portfólio**: Carrossel com projetos realizados
6. **Sobre Nós**: História da agência, missão e valores
7. **Planos e Pacotes**: Tabela comparativa de preços
8. **Blog**: Prévia de artigos sobre marketing digital
9. **Contato**: Formulário de contato e informações
10. **Footer**: Links úteis e redes sociais

## 🚀 Como Usar

1. **Instalação Local**:
   ```bash
   # Clone o repositório
   git clone [url-do-repositorio]
   
   # Navegue até a pasta
   cd conect-plus-website
   
   # Abra o index.html no navegador
   start index.html  # Windows
   open index.html   # macOS
   xdg-open index.html  # Linux
   ```

2. **Servidor Local**:
   ```bash
   # Com Python
   python -m http.server 8000
   
   # Com Node.js (npx)
   npx serve .
   
   # Com PHP
   php -S localhost:8000
   ```

3. **Deploy**:
   - Upload dos arquivos para servidor web
   - Configurar HTTPS
   - Configurar compressão GZIP
   - Configurar cache headers

## 🎨 Customização

### Cores
Para alterar as cores, modifique as variáveis no Tailwind config:
```javascript
colors: {
    primary: '#7B2CBF',    // Roxo principal
    secondary: '#FF4F79',  // Rosa coral
    accent: '#FFA552',     // Laranja pêssego
    neutral: '#F9FAFB',    // Branco gelo
    gray: '#6B7280'        // Cinza médio
}
```

### Animações
As animações podem ser ajustadas no arquivo `styles.css`:
- Duração das transições
- Efeitos de entrada
- Timing functions
- Keyframes personalizados

### Conteúdo
Para alterar textos e imagens:
1. Edite o arquivo `index.html`
2. Substitua as imagens placeholder
3. Atualize informações de contato
4. Modifique depoimentos e métricas
